# SurveyCake 貼標至社群聯絡人

### SurveyCake標籤自動匯入社群聯絡人

SurveyCake 設定之標籤（如 「題目標籤」 與 「選項標籤」 ），填完送出（被貼標）後標籤可 「自動匯入」 到他們的Omnichat社群聯絡人身上。

### 檢視問卷詳情&手動匯出問卷回應資料

1. 進入問卷詳情/明細匯出
2. 明細匯出中，確認有確實紀錄變數
3. 問卷回應資料，可匯出 .csv 檔案
4. 可將 .csv 檔案人工整理後，匯入 Omnichat 中
5. 可更新顧客之「標籤」、「自訂屬性」

{% hint style="info" %}
目前SurveyCake  「標籤」已經可以自動匯入社群聯絡人 ; 自訂屬性仍須人工匯出匯入處理。
{% endhint %}
