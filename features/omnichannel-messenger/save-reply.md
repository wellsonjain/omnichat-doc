# 預存回覆

## 使用預存回覆

按下訊息輸入區域下方的**書本按鈕**會跳出選擇預存回覆的區塊

<figure><img src="../../.gitbook/assets/截圖 2024-03-27 下午5.08.26.png" alt="" width="375"><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/截圖 2025-01-08 上午10.54.54 (1).png" alt=""><figcaption><p>預存回覆設定頁面</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/截圖 2025-01-08 上午10.55.08 (1).png" alt=""><figcaption><p>點擊 「套用至對話框」</p></figcaption></figure>

1. 可依照主題或是關鍵字去搜尋預存回覆
2. 可依照類別去搜尋預存回覆
3. 選擇想要使用的預存回覆
4. 右方可以預覽已選擇的預存回覆內容
5. 點擊 「發送」 將直接送出預存回覆內容給顧客；點擊 「套用至對話框」 可先編輯後再送出
6. 左下角 「[管理預存回覆](https://docs.omnichat.ai/features/ke-fu-she-ding/guan-li-yu-cun-hui-fu)」 可以查看／編輯／刪除／建立預存回覆
7. 點擊 「套用至對話框」 ，預存回覆內容會出現在訊息輸入框中，可先編輯後再送出

{% hint style="info" %}
此列表**只能看到適用於登入的團隊角色權限的預存回覆（特定權限 / 分店 / 只有自己），**&#x82E5;<mark style="color:red;">**發送的團隊角色如果不適用該預存回覆時，則該則預存回覆 「不會顯示」 。**</mark>

例如建立預存回覆時，適用對象選擇 『只有自己』 ，則除了自己以外沒有人看得到這則預存回覆。
{% endhint %}

### 使用含有檔案的預存回覆：

<figure><img src="../../.gitbook/assets/截圖 2024-04-10 上午11.13.38.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
* 僅 Whatsapp、Facebook對話支援使用含有檔案的預存回覆
* LINE、IG、Webchat、Wechat，不可選擇含有檔案的預存回覆，並顯示 「目標渠道不支援發送含有檔案類型的預存回覆」
{% endhint %}

### 使用含機器人的預存回覆：

<figure><img src="../../.gitbook/assets/截圖 2024-04-10 上午11.13.47.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
* 機器人類型的預存回覆只支援在以下平台使用：
  * WhatsApp
  * LINE
  * Facebook
  * Instagram
* 只能選到所適用於該對話的機器人預存回覆，不適用則將顯示 「目標渠道不支援發送此類型的機器人模組」
* 如果該機器人模組已被刪除，將顯示 「此預存回覆的機器人模組已被刪除」
* **發送含有機器人的預存回覆時，會直接發送，不可編輯模組內容**
{% endhint %}
